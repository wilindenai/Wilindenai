¿Cual es la finalidad de este Ciclo formativo?

La finalidad de este ciclo formativo de Sistemas Microinformáticos y Redes es preparar a los estudiantes para instalar, configurar, mantener y administrar sistemas informáticos.

Objetivos del ciclo formativo

  • Instalar y configurar el software básico y de aplicación.
  • Configurar redes locales cableadas, inalámbricas o mixtas.
  • Administrar redes locales.
  • Solucionar problemas técnicos.
  • Instalar, configurar y mantener servicios multiusuario.
  • Instalar, configurar y mantener dispositivos compartidos.
  • Asegurar la funcionalidad de los sistemas.
  • Aplicar protocolos de calidad, seguridad y respeto al medio ambiente.

Salidas laborales

Este ciclo formativo prepara a los estudiantes para trabajar como:
  • Técnicos de soporte informático
  • Especialistas en seguridad informática
  • Gestores de sistemas microinformáticos
  • Trabajadores autónomos
  • Empleados de empresas

Herramientas que he utilizado durante el ciclo

  • Visual Studio Code
  • Cisco Packet Tracer
  • Virtualbox junto con máquinas virtuales
  • Aplicaciones ofimáticas
  • Herramientas de sistema
