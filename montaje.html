Montaje y mantenimiento de equipos

Unidades estudiadas en este módulo y tareas realizadas

Unidades estudiadas

  • Componentes de un sistema informático.
  • Análisis del mercado de componentes de equipos microinformáticos.
  • Electricidad de los sistemas microinformáticos.
  • Riesgos laborales y protección ambiental.
  • Ensamblado de sistemas microinformáticos.
  • Mantenimiento de equipos microinformáticos.
  • Aspectos software relativos al mantenimiento de equipos.
  • Nuevas tendencias en equipos.

Tareas realizadas

Las tareas que he realizado en este módulo son:
  • Estudio en el mercado actual sobre los diferentes componentes de un sistema informático
  • Profundo análisis de los mercados para componentes microinformáticos
  • Elección y configuración de los diferentes componentes eléctricos de un sistema informático
  • Documento de análisis sobre los riesgos laborales y la protección mediambiental
  • Elección y ensamblado de los diferentes componentes de un sistema informático
  • Aprendizaje de las técnicas de mantenimiento hardware de equipos informáticos
  • Mantenimiento software de equipos informáticos

Herramientas que he utilizado durante este módulo

  • Búsqueda de información en páginas web
  • Herramientas manuales ( como destornilladores, alicates, productos de limpieza, etc, ) para el montaje y desmontaje de componentes de equipos informáticos
