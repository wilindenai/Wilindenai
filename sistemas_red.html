Sistemas operativos en red

Unidades estudiadas en este módulo y tareas realizadas


Unidades estudiadas

  • Instalación de sistemas operativos en red.
  • Active Directory. Gestión de dominios.
  • Gestión de usuarios, grupos y equipos.
  • Gestión de recursos compartidos en Windows Server.
  • Integración de Windwos Server en GNU/Linux.
  • Monitorización y uso del sistema operativo.

Tareas realizadas

Las tareas que he realizado en este módulo son:

  • Creación y eliminación de usuarios, grupos y unidades organizativas
  • Creación de plantillas
  • Creación de perfil móvil
