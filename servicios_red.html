Unidades estudiadas en este módulo y tareas realizadas


Unidades estudiadas

  • Repaso de TCP/IP y el Servicio DHCP.
  • El servicio DNS.
  • El servicio FTP.
  • Gestión de servicios de correo electrónico.
  • Gestión de servidores web.

Tareas realizadas

Las tareas que he realizado en este módulo son:
  • Renovación de ips desde comandos tanto en Windows como en Linux
  • Utilización del programa Wireshark
  • Configuración del servidor DHCP
  • Configuración y uso del DNS
  • Simulación de problemas con DNS
  • Instalar Bind9 y su comprensión.
  • Instalación y configuración del servidor FTP
  • Instalación y configuración del cliente Filezilla
  • Establecimiento de limites de conexión, cuota y ancho de banda
  • Instalación y configuración del servidor PXE
  • Instalación y configuración de MTA, MDA, MUA
  • Instalación y configuración de SpamAssasin y ClamAV
  • Configuración de Spam y Malware
  • Instalación y configuración del servidor FTP
  • Prácticas con el Servidor web Apache y el Servidor web IIS en Windows Server
  • Uso de OpenVPN
  • Configuración del Servidor SSH
  • Diferentes prácticas con Cisco Packet Tracer
